ORCA 6.0 multireference input fixtures.

Source:
https://www.faccts.de/docs/orca/6.0/manual/contents/typical/mrci.html

Only complete ORCA input examples with an explicit geometry declaration are
stored here. Narrative snippets and output-only tables are not included.

Coverage includes:
- plain MRCI and SORCI examples
- multi-job RI-MP2 to MRCI / MRACPF workflows
- MRACPF2a with DoDDCIMP2
- parameter scans through `%paras` in both `xyz` and `int` coordinate sections
